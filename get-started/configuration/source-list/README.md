# `--cfg-source-list`

Possible values for the source list are:

- `local`
- `url`
- `camera`
- `dropbox `
- `gdrive`
- `facebook`
- `gphotos`
- `instagram`
- `flickr`
- `vk`
- `evernote`
- `box`
- `onedrive`
- `huddle`

## Example

```css
.my-configuration {
  --cfg-source-list: 'url, local, instagram';
}
```
